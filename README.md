# Rise of Kingdoms AP Calculator

Single-page Next.js calculator with Turkish, English, and Spanish support.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — you will be redirected to `/tr` by default.

## Locales

| Language | URL |
|----------|-----|
| Turkish  | `/tr` |
| English  | `/en` |
| Spanish  | `/es` |

Translations live in `src/i18n/messages/`.
